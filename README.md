# JavaFX Chess Game

A chess game implementation using JavaFX, featuring:
- Complete chess rules implementation
- Graphical user interface
- Move validation
- Game state tracking

## Technologies
- Maven
- Java
- JavaFX

## Setup
1. Clone the repository
2. Install dependencies
3. Run `Main.java`

## Project Structure
```
src/
  ├── model/
  │   ├── pieces/
  │   │   └── (Chess pieces)
  │   ├── Board.java
  │   └── ChessGame.java
  ├── ui/
  │   ├── ChessBoardUI.java
  │   └── SquareView.java
  └── util/
      ├── Position.java
      └── Color.java
```
